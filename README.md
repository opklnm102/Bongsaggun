# 봉사꾼


분산되어 있는 **봉사활동 모집 정보들을 한 곳에 모아 검색**할 수 있도록 해주는 검색 플랫폼

자신에게 적합한 봉사활동을 보다 편리하게 검색할 수 있도록 `조건검색` 기능과 어떤 봉사활동이 곧 마감되고 어떤 봉사할동들이 새롭게
생겨났는지 달력을 통해 확인할 수 있는 `봉사달력` 기능을 통해 보다 쉽게 봉사를 찾고, 신청할 수 있도록 도와주는 서비스입니다.
